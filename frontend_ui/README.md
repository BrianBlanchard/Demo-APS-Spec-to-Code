# UI Storyboarding Templates

This directory contains persistent templates for frontend UI storyboarding packets.

## Structure

- `best-practices/` - Best practice guidelines for frontend development
- `INITIAL_PROMPT.md` - Initial prompt for UI storyboarding
- `extract_component_code.py` - Script for extracting component code
- `README.md` - This file

## Usage

These templates are copied into workspace-specific UI storyboarding packets on the shared Docker volume when `prepare_frontend_ui_packet_for_workspace` is called.

## Template Files

### Best Practices

- `DataFetching.md` - Guidelines for data fetching patterns
- `ProjectStructure.md` - Frontend project structure recommendations
- `StateManagement.md` - State management best practices
- `Typescript.md` - TypeScript usage guidelines

### Core Files

- `INITIAL_PROMPT.md` - Master prompt for UI storyboarding generation
- `extract_component_code.py` - Utility script for code extraction
